SF Microclimates

Real-time temperature & humidity data for 49 San Francisco neighborhoods via PurpleAir sensors

Quick Start

Get weather data for any SF neighborhood with a simple API call:

curl https://microclimates.solofounders.com/sf-weather/mission

{
  "neighborhood": "mission",
  "name": "Mission District",
  "temp_f": 71,
  "humidity": 40,
  "sensor_count": 29
}

Endpoints

/sf-weather All neighborhoods
/sf-weather/:neighborhood Single neighborhood
/neighborhoods List all available

# SF Microclimates Skill

Get hyperlocal SF weather data.

## Triggers
- "weather in [neighborhood]"
- "sf weather mission vs sunset"
- "is it foggy in the richmond?"

## Usage
```bash
# Single neighborhood
curl https://microclimates.solofounders.com/sf-weather/marina

# Compare (fetch both)
curl https://microclimates.solofounders.com/sf-weather/mission
curl https://microclimates.solofounders.com/sf-weather/sunset
```

Why Microclimates?

San Francisco is famous for its microclimates. It can be 65°F and sunny in the Mission while 52°F and foggy in the Sunset — just 3 miles apart.

This API aggregates data from 400+ PurpleAir sensors across the city, giving you real temperature readings by neighborhood — not just a single "San Francisco" number.
